PC Configurator
Веб-сервис для интеллектуального подбора комплектующих ПК.

Команда:
Сё Виталий - PM / Аналитик
Зябликов Константин - Разработчик

Для работы теста необходим node.js и модули:
npm init -y
npm install --save-dev eslint globals @eslint/js
npm install --save-dev jest

Для запуска теста:
npx jest
